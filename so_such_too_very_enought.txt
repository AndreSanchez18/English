So, such

So y such son empleados para introducir la consecuencia de algún hecho, o para enfatizar.
Del mismo modo, so también se emplea para expresiones cotidianas como: That’s so
interesting! ¡Qué interesante! 


Noun + verb + so + adjective/adverb/determiner

	My suitcase was so heavy that I couldn’t carry it.
	Mi maleta era tan pesada que no podía cargarla.
	
	It was so cold!
	¡Hacía tanto frío!
	
Such + adjective + noun




Too

Too se usa para expresar algo negativo, el individuo no está contento conla situación
en la que se encuentra inmerso. 
	
Too + adjective/adverb/determiner

	It was too cold to sleep.          ### No se usa "that" (que) siempre con el too se usa (to + verb infinito)
	Hacía demasiado frío para dormir.
	
	’m speaking too quickly. I’m sorry!
	Estoy hablando demasiado rápido. ¡Disculpa!
	
	That’s too much money. I can’t afford it.
	Es demasiado dinero. No puedo permitírmelo.

Enough

"Enough" significa "suficiente" y se usa para indicar que algo alcanza o no alcanza
un cierto nivel. Se puede usar con adjetivos, adverbios, sustantivos y verbos.

Con adjetivos/adverbios → "Adjetivo/adverbio + enough"
🔹 I'm not strong enough to lift that box. (No soy lo suficientemente fuerte para levantar esa caja.)
🔹 She didn't run fast enough to win. (No corrió lo suficientemente rápido para ganar.)

Con sustantivos → "Enough + sustantivo"
🔹 We have enough money to buy a new phone. (Tenemos suficiente dinero para comprar un nuevo teléfono.)
🔹 There aren’t enough chairs for everyone. (No hay suficientes sillas para todos.)

Con verbos → "Verb + enough"
🔹 I didn’t study enough for the test. (No estudié lo suficiente para el examen.)
🔹 She doesn’t eat enough. (Ella no come lo suficiente.)



✅ "so" → "His eyes were so bad that he couldn't read the number plate of the car in front."

📌 Explicación:
Se usa "so" antes de adjetivos (bad) para enfatizar una cualidad y suele ir seguido de "that" para indicar un resultado o consecuencia.

🔹 Ejemplo similar:
✅ She was so tired that she fell asleep immediately.
(Estaba tan cansada que se durmió de inmediato.)

🔻 Por qué no las otras opciones:
❌ "Such" → Se usa antes de sustantivos con o sin adjetivos (such a bad day).
❌ "Too" → Expresa un exceso con una connotación negativa (too bad to see anything).
❌ "Very" → Se usa para enfatizar, pero no encaja con that en esta estructura.
